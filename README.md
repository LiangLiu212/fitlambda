# fitlambda
1 dimension fit with multiple function

## Example
![product-screenshot](https://github.com/LiangLiu212/fitlambda/blob/main/example.png)

## Usage

* build the diy function
```
cd RooMyPdf
root -l
root [0] .L RooBifurStudent.cxx+
root [1] .L RooStudent.cxx+
```

## Functions

* [RooGaussian](https://root.cern.ch/doc/master/classRooGaussian.html)
* [RooBifurGauss](https://root.cern.ch/doc/master/classRooBifurGauss.html)
